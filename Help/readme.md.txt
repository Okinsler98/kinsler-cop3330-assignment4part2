Use file to save, import and import a todo list.
The directory is a where you will type the path used to save or load a todo list.

The create feature is used to add items to the current todo list.
The delete option will delete the currently selected list item.
The clear option will clear the current todo list and delete all items.

To navigate between selected items use the "previous" and "next" buttons below the the todo list.

All items can be edited by typing the the corresponding text box, but changes wont be made unless you click the "submit changes" button

The view tab will allow you to view your list according to the completion status of the items on the todo list.
